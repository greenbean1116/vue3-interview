# Table 元件練習題目

## 題目1: 基礎表格展示

**難度**: ⭐

**需求**:

- 使用 Vue3 + TypeScript 建立一個 Table 元件
- 顯示使用者清單資料 (id、姓名、信箱、年齡、城市)
- 支援自訂列設定
- 響應式樣式

**資料參考**: 使用 `mockUsers.ts` 中的使用者資料

---

## 題目2: 表格排序 + 分頁

**難度**: ⭐⭐

**需求**:

- 基於題目1，新增排序功能
  - 點擊表頭可排序 (asc/desc)
  - 支援多欄排序
- 新增分頁功能
  - 每頁顯示 10 筆資料
  - 支援跳頁、上一頁、下一頁

**資料參考**: 使用 `mockUsers.ts` 中的使用者資料 (需要50+筆)

---

## 題目3: 搜尋 + 篩選

**難度**: ⭐⭐

**需求**:

- 基於題目2，新增搜尋功能
  - 支援按姓名、信箱搜尋
  - 即時過濾結果
- 新增篩選功能
  - 按城市篩選
  - 按年齡範圍篩選
- 搜尋和篩選可同時使用

**資料參考**: 使用 `mockUsers.ts` 中的使用者資料

---

## 題目4: 表格列操作 + 表單彈窗

**難度**: ⭐⭐⭐

**需求**:

- 基於題目3，每列新增操作按鈕 (編輯、刪除)
- 編輯功能
  - 點擊編輯按鈕彈出表單彈窗
  - 編輯後更新表格資料
  - 支援驗證
- 刪除功能
  - 點擊刪除按鈕，確認後刪除該列
- 新增功能
  - 表格上方新增按鈕
  - 彈出表單，新增後加入表格

**資料參考**: 使用 `mockUsers.ts` 中的使用者資料

---

## 題目5: 列選擇 + 批量操作

**難度**: ⭐⭐⭐

**需求**:

- 基於題目4，新增列選擇功能
  - 每列前有 checkbox
  - 表頭有全選/反選按鈕
  - 選中列醒目顯示
- 批量操作
  - 選中多列後，顯示批量刪除按鈕
  - 支援批量編輯某個欄位（如：批量修改城市）
  - 顯示已選擇數量

**資料參考**: 使用 `mockUsers.ts` 中的使用者資料

---

## 加分項

如果完成上述題目，可嘗試以下功能：

- ✨ **虛擬滾動**: 表格有大量資料時使用虛擬滾動最佳化效能
- 🔄 **動態表頭**: 支援拖曳表頭調整欄寬、排序
- 📊 **資料統計**: 顯示統計資訊（總數、合計等）
- 💾 **本地儲存**: 保存使用者的排序、篩選狀態
- 📱 **響應式**: 在行動裝置顯示為卡片清單
- 🎨 **主題切換**: 支援淺色/深色主題
